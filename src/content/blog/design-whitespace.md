## 什么是留白？

留白（White Space），也称为负空间，是指设计中未被内容占据的空间。很多人误以为留白是"浪费"，实际上它是设计中最强大的工具之一。

## 留白的力量

### 1. 提升可读性

适当的留白可以让文本更容易阅读。研究表明，行高（line-height）和段落间距对阅读理解有显著影响。

```css
/* 好的排版 */
.article {
  line-height: 1.8;
  max-width: 65ch;
  padding: 2rem;
}
```

### 2. 创造焦点

通过留白的对比，我们可以引导用户的注意力到最重要的元素上。

### 3. 传达高级感

奢侈品牌往往使用大量的留白，这种"少即是多"的理念传达出产品的稀缺性和高品质。

## 实践建议

1. **不要害怕空白** - 给元素足够的呼吸空间
2. **建立系统** - 使用一致的间距单位
3. **分层级** - 重要的内容配更多的留白

## 结语

下次设计时，不妨问问自己：**如果去掉这个元素会怎样？** 如果答案是什么都不会损失，那就大胆地移除它。
